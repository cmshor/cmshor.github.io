Coloring dots!

Hello PfT workshop participants and coloring fans!

On this webpage, you can visualize dot colorings on a large (ish) scale. We're working in an n by n lattice along with a prime q. We have the function tq(m) which gives the highest power of q that divides the positive integer m. We also define tq(0) to be infinity.

For each lattice point P=(x,y), we calculate A=tq(x), B=tq(y), and C=tq(n).

If A is smallest, color the point blue. If B is smallest, color the point green. If C is smallest, color the point red.

In the case of a tie involving A, the winner is A. In case of a tie between B and C, the winner is B.

Click the Initialize button to get started.

Note: It's easy to break this! If q isn't prime, or if n isn't a positive integer, you'll get an error message. Choosing appropriate numbers should fix things. If the numbers get too large, it may get stuck. You can hit the Initialize button to restart.
